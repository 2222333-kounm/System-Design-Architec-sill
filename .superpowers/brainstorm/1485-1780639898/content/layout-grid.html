✦ grid-template-areas
折叠可选，文本输入 CSS 原生 area 语法，上方显示可视化区域布局示意
📋 输出格式 
{
  type: 'layout',
  css: {
    display: 'grid',
    gridTemplateColumns: '1fr 1fr 1fr',
    gridTemplateRows: 'auto auto',
    gap: '16px',
    rowGap: '16px',
    columnGap: '16px',
    justifyItems: 'center',
    alignItems: 'center',
    justifyContent: 'center',
    alignContent: 'center',
    gridAutoFlow: 'row',
    // 可选:
    gridTemplateAreas: "'h h h' 's c c' 'f f f'"
  },
  extra: {
    mode: 'grid',
    columns: ['1fr','1fr','1fr'],
    rows: ['auto','auto'],
    children: []
  }
}
