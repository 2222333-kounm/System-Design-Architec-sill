循环检测 · 拓扑排序 · 多输出 · 错误处理

循环依赖检测

输出节点连回色块 → 检测到环 → 弹窗警告 ❌ 检测到循环依赖，连接已被阻断，红色高亮显示违规连线。

①

拓扑排序执行

自动计算节点执行顺序，上游先执行 → 下游后执行。修改任意节点 → 只重算受影响的下游路径，不做全局重算。

⊕

多输出支持

统一数据格式：{ type, css, extra }
合并节点可通过 extra 携带原始数据副本，不拆端口。

⚠

错误处理

节点计算失败 → 输出 { type: 'error', message }
上游出错时下游显示占位（灰色虚线框）并保持最后有效状态。

端口类型匹配规则

端口类型 可连接 禁止连接
color→ color, css→ image, text
image→ image, css→ color, text
text→ text, css, any→ image
css→ css, any→ color, image
interactive→ interactive, css→ color, image
any→ 任意端口—
