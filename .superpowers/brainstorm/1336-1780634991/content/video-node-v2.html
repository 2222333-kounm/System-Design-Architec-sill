📋 统一数据格式
type css extra

所有节点统一使用 { type, css, extra } 格式输出：

  • type — 节点类型标识 ('color', 'text', 'image' 等)
  • css — 仅含布局属性，下游读此即可排版
  • extra — 非布局特有数据（播放行为、替代文本等）
🎬 视频节点输出 
{
  type: 'video',
  css: { aspectRatio: '16/9' },
  extra: {
    src: '...',
    autoplay: true,
    loop: true,
    muted: false,
    controls: true,
    poster: '...'
  }
}
