📋 统一数据格式规范

所有节点共用 { type, css, extra } 结构

核心结构

type

string

节点类型标识。
如 'color', 'text', 'image', 'video', 'css' 等。

css

Record<string, string>

仅包含直接影响布局的样式属性。
下游节点读取 css 即可完成排版占位。

extra

object

节点特有的非布局数据。
输出节点需要渲染时读取 extra。

各节点输出格式

🎨 色块 
{
  type: 'color',
  css: {
    background: 'rgba(0,113,227,1)',
    width: '320px',
    height: '120px',
    borderRadius: '12px',
    opacity: '1'
  }
}
📝 文字 
{
  type: 'text',
  css: {
    fontFamily: 'PingFang SC',
    fontSize: '16px',
    fontWeight: '400',
    lineHeight: '1.5',
    letterSpacing: '0em',
    color: '#1D1D1F',
    textAlign: 'left'
  },
  extra: {
    content: '示例文字',
    textDecoration: 'none',
    textTransform: 'none'
  }
}
🖼️ 图片 
{
  type: 'image',
  css: {
    objectFit: 'cover',
    backgroundRepeat: 'no-repeat',
    backgroundColor: '#374151',
    backgroundImage: 'url(...)'
  },
  extra: {
    alt: '图片描述',
    loading: 'lazy',
    renderMode: 'background'
  }
}
🎬 视频 
{
  type: 'video',
  css: { aspectRatio: '16/9' },
  extra: {
    src: '...',
    autoplay: true,
    loop: true,
    muted: false,
    controls: true,
    poster: '...'
  }
}
🔘 按钮 
{
  type: 'interactive',
  css: {
    display: 'inline-block',
    padding: '8px 16px',
    background: '#0071E3',
    color: '#FFFFFF',
    borderRadius: '980px',
    border: 'none',
    cursor: 'pointer',
    fontSize: '14px'
  },
  extra: {
    kind: 'button',
    text: '立即购买',
    hover: { background: '#0077ED' },
    active: { background: '#0068D9' }
  }
}
