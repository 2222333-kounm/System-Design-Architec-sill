## 1. 基础组件（6 个）

### 1.1 色块
- **说明：** 纯色/渐变背景
- **输入：** color（可接收上游色值）
- **输出：** color-block（含完整 CSS）
- **参数：** 色值（color picker）、宽度、高度、圆角、不透明度

### 1.2 文字
- **说明：** 字体、字号、行高、字重
- **输入：** any（可接收文字内容）
- **输出：** text（含 CSS 样式）
- **参数：** 内容、字体家族、字号、字重、行高、字距、颜色、对齐

### 1.3 图片
- **说明：** 图像填充/适配
- **输入：** any（可接收图片 URL）
- **输出：** image（含 CSS 样式）
- **参数：** 图片 URL、object-fit、background-repeat

### 1.4 视频
- **说明：** 视频媒体元素
- **输入：** any（可接收视频 URL）
- **输出：** video（含 video 标签属性）
- **参数：** 视频 URL、自动播放、循环、静音、控件

### 1.5 按钮
- **说明：** 含状态（默认/hover/按压）
- **输入：** text（按钮文字）
- **输出：** interactive（三态样式）
- **参数：** 文字、常态色、hover 色、active 色、圆角、内边距

### 1.6 图标
- **说明：** 矢量或字体图标
- **输入：** any
- **输出：** css（含 font-size/color/opacity）
- **参数：** 图标选择、大小、颜色、透明度

## 2. 布局 & 结构（4 个）

### 2.1 布局容器
- **说明：** Flex / Grid，控制方向、对齐、间距
- **输入：** css（多个子元素）
- **输出：** css（容器样式）
- **参数：** display（flex/grid）、方向、主轴对齐、交叉轴对齐、间距 gap、flex-wrap

### 2.2 间距节点
- **说明：** Padding / Margin 独立控制
- **输入：** css
- **输出：** css（带间距后的样式）
- **参数：** padding-top/right/bottom/left、margin-top/right/bottom/left（每个方向独立控制）

### 2.3 组件实例
- **说明：** 复用组件抽象，支持引用
- **输入：** any（多个输入，对应组件插槽）
- **输出：** css（组件实例的最终样式）
- **参数：** 组件引用 ID、实例名称、插槽映射

### 2.4 断点
- **说明：** 媒体查询，不同视口样式变化
- **输入：** css（桌面端样式）
- **输出：** css（带媒体查询包裹的样式）
- **参数：** 断点名称（sm/md/lg/xl）、视口宽度值、覆盖样式（多个）

---

## 3. 变换 & 特效（6 个）

### 3.1 变换
- **说明：** transform（平移/旋转/缩放）
- **输入：** css
- **输出：** css（含 transform 属性）
- **参数：** 缩放、旋转角度、透明度、圆角

### 3.2 蒙版
- **说明：** 遮罩/剪裁
- **输入：** image
- **输出：** image（含 mask 信息）
- **参数：** 形状（矩形/圆形/渐变）、羽化、反转

### 3.3 边框
- **说明：** 边框宽度、样式、圆角
- **输入：** css
- **输出：** css（含 border 属性）
- **参数：** 线型（实线/虚线/点线）、粗细、颜色、圆角

### 3.4 阴影
- **说明：** 盒阴影/文字阴影
- **输入：** css
- **输出：** css（含 box-shadow）
- **参数：** 类型（投影/内阴影）、X 偏移、Y 偏移、模糊、扩展、颜色

### 3.5 鼠标跟随
- **说明：** 鼠标交互特效
- **输入：** css
- **输出：** interactive（含鼠标跟随配置）
- **参数：** 效果（视差/发光/3D 倾斜）、强度、范围

### 3.6 转场
- **说明：** 过渡动画
- **输入：** css
- **输出：** interactive（含过渡配置）
- **参数：** 触发（hover/click）、变换类型（缩放/位移/淡入）、时长、缓动曲线

---

## 4. 工具 & 全局（4 个）

### 4.1 转换
- **说明：** 数据类型转换
- **输入：** any
- **输出：** any（转换后的类型）
- **参数：** 转换模式（text→css / color→css / image→css / css→text / object→json / json→object）

### 4.2 合并
- **说明：** 合并多个节点输出
- **输入：** css × 3（A/B/C 三路输入）
- **输出：** merged
- **参数：** 模式（叠加/堆叠/覆盖）

### 4.3 全局 Token
- **说明：** 设计令牌（颜色/字体/间距变量）
- **输入：** 无
- **输出：** tokens（所有 CSS 变量集合）
- **参数：** 无（自动读取 tokens.css）

### 4.4 输出
- **说明：** 最终结果出口
- **输入：** css
- **输出：** 无（数据推送到右侧预览面板）
- **参数：** 无
