# 无限画布节点编辑器 — 设计规范

> **状态：** 临时草稿，待审核
> **目的：** 将设计系统从被动展示升级为可组合的生产工具
> **引擎：** litegraph.js（ComfyUI 同款）
> **风格：** ComfyUI 节点风格 · 左入右出端口 · 暗色毛玻璃主题最后适配

---

## 0. 核心设计原则

### 端口规则

| 规则 | 普通节点 | 合并节点（特殊） |
|------|---------|----------------|
| 输入端 | **1 个**，最多接受 **1 个上游连接** | **多个独立输入端口**，每个端口最多接受 **1 个上游连接** |
| 输出端 | **1 个**，可连接**多个下游** | **1 个**，可连接多个下游 |
| 多节点连到同一输入 | UI 阻止或自动替换已有连接 | 不适用（每个端口独立） |
| 合并模式 | 不适用 | 用户在节点内选择：**叠加 / 堆叠 / 覆盖** |

### 类型匹配规则

- 端口必须有类型，不同类型不能连接
- 允许少量安全自动转换：数字↔字符串、颜色↔CSS 字符串
- 复杂转换（对象↔数组、对象↔JSON 字符串等）要求用户插入显式的**转换节点**
- `any` 类型可连接任何类型，接收方按实际类型解析

### 数据流规则

- 系统不做隐式数据合并。多路数据需通过**合并节点**汇合
- 禁止形成循环连接（DAG 约束）

### 节点数据输出规则

- 每个节点按其主功能输出对应类型（text / color / image / css / interactive 等）
- CSS 样式信息作为数据对象的一个子字段携带
- 输出节点收到的是层层转换后的最终数据

## 1. 端口类型系统

所有节点共用一套端口颜色编码：

| 端口颜色 | 类型名 | 数据类型 | 用途 |
|---------|-------|---------|------|
| ○ 蓝 `#3B82F6` | `color` | CSS 颜色字符串 | 色值 (#hex / rgba) |
| ○ 绿 `#22C55E` | `image` | URL/DataURL 字符串 | 图片/视频数据 |
| ○ 紫 `#A855F7` | `text` | 纯文本字符串 | 文字内容 |
| ○ 橙 `#F97316` | `css` | 完整 CSS 对象（JSON） | 样式属性集合 |
| ○ 青 `#06B6D4` | `number` | 数值 | 数字参数 |
| ○ 灰 `#6B7280` | `any` | 任意类型 | 通用传递 |
| ○ 粉 `#EC4899` | `interactive` | 交互行为描述对象 | hover/click 事件 |

## 2. Token 同步机制

### 控件默认值来源

所有节点的控件默认值**从 tokens.css 读取**，而非硬编码固定值。

### 锁定/解锁机制

每个从 Token 读取默认值的控件，旁边显示一个 **🔗 链接图标**：

```
色值: [#3B82F6] 🔗         ← 锁定状态（跟随 Token）

用户修改数值后:
色值: [#FF0000] 🔗          ← 自动断开，变为灰色或断开图标
         [重置为 Token 默认值]
```

| 状态 | 图标 | 行为 |
|------|------|------|
| **锁定** | 🔗 实心连接 | 控件值跟随 Token。Token 变化时控件自动更新 |
| **解锁** | ⛓️ 断开连接 | 用户手动修改值后自动解锁。Token 变化不影响当前值 |

- 用户修改数值 → 锁自动断开 → 提示"已断开与 Token 的同步"
- 提供"重置为 Token 默认值"按钮 → 重新锁定并拉取当前 Token 值
- Token 节点本身的值变更会广播通知所有锁定的控件自动刷新

### @token 引用语法

当用户想在控件中使用 Token 引用而非手动输入时：

```
色值: [@color-primary-500] @🔗   ← @ 按钮弹出 Token 列表
```

| 功能 | 行为 |
|------|------|
| **@ 按钮** | 点击弹出 Token 选择列表 |
| **列表显示** | 默认显示**精选集**（四大类常用项），底部有"显示全部"入口 |
| **选中填充** | 选中后输入框显示 `@color-primary-500`，附带 Token 图标高亮表示引用 |
| **手动覆盖** | 用户删除 `@` 前缀 → 转为手动输入的值 |
| **解析** | 节点执行时自动将 `@xxx` 替换为 tokens.css 中对应变量值 |

### 全局 Token 节点展示规则

| 功能 | 行为 |
|------|------|
| **默认视图** | 精选子集（颜色/排版/间距/圆角四大类最常用项） |
| **展开全部** | 节点底部有"显示全部"按钮，展开后展示 tokens.css 的所有变量 |
| **搜索过滤** | 展开状态下支持关键词搜索快速定位 |
| **自定义** | 允许用户自定义"我的常用 Token"列表（后续版本） |

## 3. 数据传递格式

每个节点按其主功能输出对应类型，CSS 样式作为子字段携带。

### 类型数据示例

**文字节点输出 (text 类型)：**
```json
{
  "type": "text",
  "content": "这是一段示例文字",
  "css": {
    "font-family": "PingFang SC",
    "font-size": "24px",
    "font-weight": 600,
    "line-height": 1.5,
    "letter-spacing": 0,
    "color": "#1D1D1F",
    "text-align": "left"
  }
}
```

**色块节点输出 (color 类型)：**
```json
{
  "type": "color",
  "color": "#0071E3",
  "css": {
    "background": "#0071E3",
    "width": "320px",
    "height": "120px",
    "border-radius": "12px",
    "opacity": 1
  }
}
```

**图片节点输出 (image 类型)：**
```json
{
  "type": "image",
  "url": "https://example.com/photo.jpg",
  "css": {
    "background-image": "url(https://example.com/photo.jpg)",
    "object-fit": "cover",
    "background-repeat": "no-repeat"
  }
}
```

```
编组 ⑯ ──────────→ 画板 ⑭ ─────────────────┤
布局容器 ⑬ ──────→ 画板 ⑭ ─────────────────┤
滚动容器 ⑮ ──────→ 画板 ⑭ ─────────────────┘

转换节点 ⑰ — 插入在类型不匹配的两个节点之间
合并节点 ⑱ — 多路数据需要汇合时插入

全局 Token ⑲ ──→ (所有节点的 @xxx 引用 / 🔗 锁定引用)
```

**连线规则：**
- 类型不同不能连接。允许少量安全自动转换（数字↔字符串、颜色↔CSS 字符串）
- 复杂转换需要插入**转换节点**
- 输出端口可连接多个输入端口（一对多）
- 输入端口只接受一根连线。多路数据需通过**合并节点**汇合
- 禁止形成循环连接（DAG 约束）

## 6. 浮层集成

```
┌─────────────────────────────────────────┐
│  控制栏 [🔌 节点编辑]                     │
├─────────────────────────────────────────┤
│  ░░░ 展示卡片（背景层） ░░░               │
│  ░░░ 色板/排版/间距     ░░░               │
│                                         │
│  ┌── 浮层 ───────────────────────────┐  │
│  │ [🔌 Node Editor] [搜索节点…] [✕] │  │
│  │ ┌──────────────────────────────┐  │  │
│  │ │  LiteGraph Canvas            │  │  │
│  │ │  色块──→边框──→阴影──→输出   │  │  │
│  │ └──────────────────────────────┘  │  │
│  └───────────────────────────────────┘  │
└─────────────────────────────────────────┘
```

- 点击「🔌 节点编辑」→ 浮层弹窗（iframe 加载 node-editor.html）
- 点击 ✕ 或 ESC → 浮层关闭
- 背景展示卡片无损保留
- 节点编辑器中修改的内容实时反映在输出节点预览中
